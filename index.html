Introduction

The Chomsky hierarchy classifies languages based on their generative grammars. It consists of four levels:

  • Type 0: Unrestricted Grammar
  • Type 1: Context-Sensitive Grammar
  • Type 2: Context-Free Grammar
  • Type 3: Regular Grammar

Type 0: Unrestricted Grammar

Recognized by Turing machines.

Type 1: Context-Sensitive Grammar

Generated by context-sensitive grammars.

Type 2: Context-Free Grammar

Generated by context-free grammars.

Type 3: Regular Grammar

Generated by regular grammars.
